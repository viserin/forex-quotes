== Description ==
<p>Forex Quotes is an open source plugin that displays FX rates on your website from your trading platform. We currently offer connectivity to MT4. If you do not have a trading platform, we offer rates from a third party. Currency pairs and UI colors are customizable from the admin interface.</p>

== Installation ==
1. Install the plugin or unzip the downloaded files to the plugins folder
2. Activate the plugin in the Plugins menu
3. Go on the plugin settings page and set your preferences
4. Use [forex-quotes type="ticker"] or [forex-quotes type="table"] shortcodes in any post or page you want
5. Alternatively, instead of shortcodes, you can use it as a widget in the widget section of your admin area
